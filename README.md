# CBSOT Project 2

## Overview

CBSOT Project 2 consists of two data science and information retrieval tasks implemented using Python and Jupyter Notebook.

The project demonstrates practical applications of exploratory data analysis (EDA) and search engine implementation, covering data preprocessing, visualization, and information retrieval concepts.

---

## Project Components

### 1. Exploratory Data Analysis (EDA)

The EDA notebook focuses on understanding and analyzing the dataset through various statistical and visualization techniques.

#### Features

- Data loading and preprocessing
- Handling missing values
- Descriptive statistics
- Data visualization
- Correlation analysis
- Insight generation

---

### 2. Search Engine

The Search Engine notebook demonstrates the implementation of a simple search engine capable of retrieving relevant information from a collection of documents.

#### Features

- Text preprocessing
- Tokenization
- Stop-word removal
- Search query processing
- Document retrieval
- Basic information retrieval concepts

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Installation

Clone the repository:

```bash
git clone https://github.com/vinayak15khandelwal/CBSOT-Project-2.git
cd CBSOT-Project-2
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open either:

- `EDA.ipynb`
- `Search_Engine.ipynb`
